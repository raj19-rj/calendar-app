# Interactive Calendar Component

A polished, interactive wall calendar built with React and Next.js.

## Features

- Wall calendar aesthetic with a dynamic hero image per month
- Day range selector with clear visual states for start, end, and in-between dates
- Integrated notes section — attach notes to any selected date or range
- Notes persist using localStorage — no backend needed
- Fully responsive — side-by-side on desktop, stacked on mobile
- Month navigation with unique image per month

## Tech Stack

- Next.js 16
- React
- Tailwind CSS

## Run Locally

```bash
git clone https://github.com/raj19-rj/calendar-app.git
cd calendar-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
